## Анализ продаж веломагазинов

**Выполненные шаги:**

1. Открыть датасет с помощью `pandas`
2. Показать первые и последние 5 элементов датасета
3. Вывести информацию по датасету и типам данных
4. Указать существующую метрику
5. Для поля `Order Quantity` найти количество элементов, среднее значение, минимум, максимум, стандартное отклонение и медиану
6. Для поля `Order Quantity` построить гистограмму
7. Посчитать количество продуктов каждого типа и построить круговую диаграмму
8. Вывести все локации, где происходила продажа (страна и штаты)
9. Вывести все продукты компании
10. Вывести столбчатую диаграмму 10 самых продаваемых продуктов
11. Вывести все продажи в штате `Montana`
12. Вывести среднюю выручку с продаж клиентам какой-либо возрастной группы
13. Вывести графики по продажам в год и месяц
14. Вывести график соотношений `Unit_Cost` и `Unit_Price`
15. Построить линейную регрессию методом наименьших квадратов
16. Сравнить с линейной регрессией из пакета `scikit-learn`
17. Построить график продаж по времени
